# Ahorcado

### Descripción

El usuario debe adivinar la palabra escondida. Por cada error se mostrará visualmente los intentos fallados. Por cada fallo se agregará una parte al ahorcadito.

### Habilidades aprendidas

- Strings
- Array
- Convert
